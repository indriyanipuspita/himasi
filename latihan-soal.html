Mata Kuliah

Algoritma & Pemrograman

Jumlah Soal

30 Soal (Hal 1: 1-15)

Referensi

Modul Universitas Terbuka

#1

Manakah yang termasuk struktur dasar algoritma?

Lihat Jawaban

Jawaban: A. Runtunan, Pemilihan, Pengulangan

Tiga struktur dasar algoritma adalah Sequence (runtunan), Selection (pemilihan), dan Repetition (pengulangan).

#2

Simbol flowchart berbentuk belah ketupat digunakan untuk...

Lihat Jawaban

Jawaban: C. Decision (Keputusan)

Belah ketupat melambangkan pengambilan keputusan (Decision) yang biasanya menghasilkan output YA atau TIDAK.

#3

Tipe data yang digunakan untuk menyimpan nilai logika benar (true) atau salah (false) adalah...

Lihat Jawaban

Jawaban: C. Boolean

Boolean adalah tipe data primitif yang hanya memiliki dua kemungkinan nilai: true atau false.

#4

Manakah penulisan variabel yang valid dalam Java?

Lihat Jawaban

Jawaban: C. _namaVariabel

Identifier boleh dimulai dengan huruf, underscore (_), atau tanda dolar ($). Tidak boleh dimulai dengan angka atau spasi.

#5

Operator aritmatika modulo (%) digunakan untuk...

Lihat Jawaban

Jawaban: B. Mendapatkan sisa bagi

Operator Modulo (%) menghasilkan sisa pembagian dari dua bilangan operandnya.

#6

Jika a = 5 dan b = 2, berapakah hasil dari (a > b) && (b < 0)?

Lihat Jawaban

Jawaban: B. False

(5 > 2) adalah True. (2 < 0) adalah False. True AND False menghasilkan False.

#7

Struktur 'if-else' termasuk dalam...

Lihat Jawaban

Jawaban: B. Branching (Percabangan)

If-else digunakan untuk mengarahkan alur program ke cabang kode tertentu berdasarkan kondisi boolean.

#8

Pernyataan 'switch-case' paling cocok digunakan ketika...

Lihat Jawaban

Jawaban: C. Memilih satu dari banyak opsi nilai pasti (fixed)

Switch-case efektif untuk percabangan multi-arah berdasarkan kesamaan nilai (equality) variabel tunggal.

#9

Perulangan 'for' biasanya digunakan ketika...

Lihat Jawaban

Jawaban: B. Jumlah perulangan sudah diketahui pasti

For loop ideal untuk iterasi yang jumlah perulangannya (count-controlled) sudah ditentukan sejak awal.

#10

Apa output dari: int i=0; while(i<3) { print(i); i++; }

Lihat Jawaban

Jawaban: C. 012

Loop berjalan saat i=0, i=1, i=2. Saat i=3 kondisi i<3 bernilai salah dan loop berhenti.

#11

Array adalah struktur data yang menyimpan elemen dengan tipe data...

Lihat Jawaban

Jawaban: B. Sama (Homogen)

Array menyimpan kumpulan elemen yang bertipe data sama dalam urutan memori yang kontigu.

#12

Indeks pertama dari sebuah array biasanya dimulai dari angka...

Lihat Jawaban

Jawaban: C. 0

Dalam bahasa C, C++, Java, dan banyak bahasa modern lainnya, array menggunakan zero-based indexing.

#13

Fungsi (function) atau method yang tidak mengembalikan nilai menggunakan keyword...

Lihat Jawaban

Jawaban: D. void

Keyword void menandakan bahwa sebuah method melakukan tugas tertentu tanpa mengembalikan nilai (return value) ke pemanggilnya.

#14

Parameter yang dikirim saat pemanggilan fungsi disebut...

Lihat Jawaban

Jawaban: B. Parameter Aktual (Argumen)

Parameter Aktual adalah nilai nyata yang dikirimkan ke fungsi saat dipanggil, sedangkan Parameter Formal adalah variabel dalam definisi fungsi.

#15

Algoritma pencarian yang bekerja dengan membagi data menjadi dua bagian secara berulang disebut...

Lihat Jawaban

Jawaban: C. Binary Search

Binary Search mencari data pada array terurut dengan membagi interval pencarian menjadi dua bagian setiap iterasinya.
